## 🛠 Fungsi Kontrak
Kontrak ini memiliki 5 fungsi utama:
1. `create_book`: Menambah buku baru.
2. `get_books`: Mengambil semua daftar buku.
3. `get_book_by_id`: Mencari buku berdasarkan ID unik.
4. `update_book`: Mengubah data buku yang sudah ada.
5. `delete_book`: Menghapus buku dari penyimpanan.
